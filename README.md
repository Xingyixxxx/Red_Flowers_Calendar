# 🌺 Red Flowers (小红花花园)

> “送你一朵小红花。”

这是一个简单而温馨的**生活记录工具**。

## ✨ 关于这个项目

这个项目的灵感来源于我和女朋友恋爱过程中的一个小想法。

我们想找一种有趣的方式来记录生活中的点点滴滴和每一个值得纪念的时刻。比起枯燥的文字列表，我们觉得如果能把这些时刻变成一朵朵盛开的“小红花”，种在一个属于我们的“私家花园”里，那该是一件多浪漫的事啊。

于是，就有了这个项目。它不复杂，但每一朵花都代表着一份珍贵的回忆。

## 🎨 功能预览

* **日历花园**: 月历不再是冷冰冰的数字，而是一片待耕耘的土地。
* **种下美好**: 点击任意一天，就能“种”出一朵小红花（带有可爱的生长动画）。
* **时光统计**: 自动统计当你月和总共收集了多少朵小红花。
* **数据持久化**: 所有记录都会自动保存，不会丢失。

## 🛠️ 技术栈

* **后端**: Python (FastAPI) - 轻量、快速。
* **前端**: HTML5 + CSS3 + Vanilla JS - 纯粹的 Web 技术，为了实现那个弹跳的动画效果。
* **数据**: JSON - 简单直接的文件存储。

## 🚀 如何运行

### 1. 环境准备

确保你的电脑上安装了 Python 3.x。

### 2. 安装依赖

```bash
# 如果使用 uv (推荐)
uv sync

# 或者使用传统的 pip
pip install fastapi uvicorn jinja2
```

### 3. 启动花园

Mac/Linux 用户可以直接运行脚本：

```bash
./run_app.sh
```

或者手动运行：

```bash
uv run uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 4. 访问

打开浏览器访问：`http://localhost:8000`

## 📱 手机远程访问 (可选)

如果你想在手机上查看这个花园：

1. **局域网模式**:
    确保手机和电脑在同一 WiFi 下，访问 `http://电脑IP:8000` (例如 `192.168.1.5:8000`)。

2. **公网模式 (推荐)**:
    使用 Cloudflare Tunnel 进行内网穿透：

    ```bash
    cloudflared tunnel --url http://localhost:8000
    ```

    复制生成的链接即可在任何地方访问。
